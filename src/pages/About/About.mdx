### Redux Toolkit
Used in this example to manage the global state, consisting of the
dark mode setting and the fetching of poems.

```ts
const poemsSlice = createSlice({
  name: 'poems',
  initialState: poemsInitialState,
  reducers: {
    getPoemsStart: startLoading,
    getPoemsSuccess(state, { payload }: PayloadAction<PoemsResult>) {
      const { poems } = payload;
      state.poems = poems;
      state.isLoading = false;
      state.error = null;
    },
    getPoemsFailure: loadingFailed,
  },
});
```
